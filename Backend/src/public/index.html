API overview

Information about the routes that will be invoked to perform facial recognition.

You must request a token to perform all operations

Route to request the Http-Post token to

"../faceRecognition/signin"
Send in the json body with username and password.

Example

{
   "username": "ivan",
   "password": "admin237"
}

Headers:
   [{"Content-Type", "value": "application / json"}]
-----------------------	

Responce 200 ok:
 {
    "auth": true,
    "token": "eyJhbGc.."
}

You must request a faceRecognition to perform all operations

Route to request the Http-Post faceRecognition

"../faceRecognition"
Send in the body a Json with the information of the images that you want to analyze.

Example

	{ 
	"referencePhotos": [{
           "url": "URL_IMAGE1"},
           {"url": "URL_IMAGE2"
    }],
       "samplePhoto":"URL_IMAGE_PROFILE",
       "sampleInfo": {
            "id": 1,
            "date": "19.10.2019 20:23:15",
            "colorCode": "green"
             }
	 }
	
      Headers:
        [{"Content-Type", "value": "application / json"}]
		[{Authorization","value":"bearer hash of token"}]
		-----------------------

      Responce 200 ok:
       {
          "sampleID": "1",
      	  "result": "FAILED or PASSED"
      	}
